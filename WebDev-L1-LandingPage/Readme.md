# 🍕 PizzaCraft - Responsive Landing Page

## 📌 Project Overview

PizzaCraft is a modern, responsive landing page developed using HTML5 and CSS3. The project showcases a fictional pizza brand with an attractive user interface, responsive layout, and clean design principles.

---

## 🚀 Features

- Sticky Navigation Bar
- Hero Section with Call-to-Action Button
- Features Section
- About Section
- Customer Testimonials
- Responsive Design using Flexbox
- Modern Color Palette
- Footer with Contact Information

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Flexbox
- Google Fonts
